## Описание компонентов

### Card
Компонент для отображения информационных карточек с теоретическим материалом.
- Поддерживает заголовки, текст и изображения
- Адаптивный дизайн
- Визуальное выделение важной информации

### Quiz
Интерактивный компонент для проверки знаний.
- Множественный выбор ответов
- Моментальная обратная связь
- Подсчет правильных ответов
- Объяснение правильных ответов

### Checklist
Компонент для пошаговых инструкций и чек-листов.
- Отметка выполненных шагов
- Сохранение прогресса
- Визуальная индикация завершенности

### DiagramViewer
Просмотрщик диаграмм и схем.
- Масштабирование изображений
- Интерактивные подсказки
- Поддержка различных форматов

## Порядок изучения материала

1. **Урок 1: Подготовка клиента к блондированию**
   - Изучите теоретический материал в `src/content/theory/lesson-1.md`
   - Пройдите интерактивные тесты
   - Выполните чек-лист подготовки
   - Ознакомьтесь с диаграммами

2. **Практическое применение**
   - Используйте чек-листы во время работы
   - Обращайтесь к теории при необходимости
   - Проверяйте знания с помощью Quiz

3. **Дополнительные материалы**
   - Оригинальные файлы уроков находятся в `content/raw/`
   - Используйте их для более глубокого изучения

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build
```

## Технологический стек

- **Vite** - быстрый сборщик и dev-сервер
- **React** - библиотека для создания интерфейсов
- **Tailwind CSS** - utility-first CSS framework
- **Markdown** - для теоретических материалов
